11.1 Interface de Rede

Toda a gerência do servidor de execução é feita remotamente através da IDE do MPA. Por esta razão é necessário definir uma interface de rede adequada a essa comunicação. Isso é feito através dos parâmetros host e port.

O parâmetro host define a interface de rede à qual o servidor deve ser associado, ou seja, o endereço IP ou nome de uma das interfaces de rede disponíveis na máquina servidora. O parâmetro port define a porta dessa interface de rede que o executor deve disponibilizar para conexões de IDEs MPA. Por exemplo, para permitir a conexão através da interface de rede com endereço 10.20.30.40 através da porta 8080 pode-se utilizar o seguinte comando:

mpa-run.exe --host 10.20.30.40 --port 8080

Ou adicionar ao arquivo de parâmetros de configuração o seguinte:

HOST = ”10.20.30.40”;

PORT = 8080;

Note que o valor do parâmetro port no arquivo de parâmetros não vem entre aspas, pois como o valor é numérico esse recurso não se faz necessário. Caso nenhum valor seja especificado para o parâmetro host, seu valor default é localhost. O valor default do parâmetro port é 9090.
