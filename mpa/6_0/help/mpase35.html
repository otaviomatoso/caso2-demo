B.1 Como Funciona

O Graphviz é uma ferramenta que gera, como o nome já sugere, a visualização de grafos, no caso, grafos de ligação. Um Grafo de ligação é uma representação gráfica de estruturas que estão, de alguma maneira, interligadas.

Utilizando este conceito e as definições dos elementos de uma configuração de planta foi criada uma abstração gráfica para os atributos das configurações, sejam esses numéricos, lógicos, textuais, outras configurações ou até atributos em branco ou mal configurados. Após criar a visualização, o usuário pode salvar ou imprimir a imagem gerada para referências futuras.
