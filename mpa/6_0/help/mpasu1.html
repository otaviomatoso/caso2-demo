3.2.1 Menu Arquivo

No menu Arquivo se encontram os seguintes comandos:

Item Novo... Cria um novo projeto de aplicação MPA, ou um dos diferentes tipos de arquivos que compõem um projeto de aplicação e que são editados através da IDE. Esses arquivos podem ser uma planta de equipamentos, uma biblioteca de diagramas que especificam manobras de operação, ou ainda extensões de classes de equipamentos. Nos capítulos seguintes são apresentados o papel de cada um desses arquivos na composição da aplicação, assim como os recursos da IDE para edição deles.

Item Abrir... Permite abrir um novo projeto de aplicação, que é dado por um diretório que contenha todos arquivos que compõem a aplicação.

Item Salvar Todos Salva as alterações em todos os arquivos do projeto de aplicação atual.

Item Atualizar Recarrega os arquivos do projeto a partir de suas versões encontradas no disco.

Item Importar... Permite incorporar ao projeto de aplicação atual uma cópia de um arquivo existente no sistema. O arquivo é importado como um novo arquivo não salvo. Para persistir o arquivo no diretório do projeto é necessário salvá-lo.

Item Imprimir... Permite imprimir o arquivo sendo editado na área de edição. Nem todos os editores dão suporte a impressão.

Item Documentar... Gera uma documentação em HTML da aplicação. Na versão atual apenas a documentação de diagramas é gerada.

Item Exportar Aplicação... Seleciona uma pasta no disco para exportação da aplicação atual em arquivos que podem ser carregados na inicialização do executor do MPA. Dessa forma, o executor pode comandar a execução da aplicação durante a sua inicialização sem necessidade de enviar ou iniciar a aplicação através da IDE. Por padrão todos os documentos do projeto de aplicação são exportados. Caso um desses documentos não deva fazer parte da aplicação exportada, o mesmo deve ser desbilitado usando o menu de contexto sobre o documento ou na configuração do arquivo de projeto correspondente.

Item Executar Script... Permite a execução de scripts pré-definidos de manipulação dos dados da aplicação, assim como execução de novos scripts descritos em arquivos. Esses scripts podem gerar relatórios e outros documentos com base nos dados da aplicação sendo desenvolvida.

Item Sair Sai do aplicativo da IDE do MPA, dando a opção, quando necessário, de salvar os arquivos de projeto modificados.
