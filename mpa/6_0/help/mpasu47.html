C.1.4 Tipos Lista

É possível definir valores que são listas de valores de um determinado tipo. Para tanto basta definir o campo type (de um atributo, parâmetro ou valor de retorno de uma função ou operação) com o identificador do tipo dos valores da lista acrescido do sufixo []. Por exemplo, uma lista de valores reais é identificada pelo tipo REAL[]. Uma lista de pontos lógicos é identificada pelo tipo BOOLEAN_POINT[]. Também é possível definir listas de equipamentos como por exemplo ValvulaOnOff[] define uma lista de equipamentos da classe ValvulaOnOff. Não é possível ter listas contendo valores de tipos diferentes.

Uma lista é representada por uma tabela Lua contendo uma lista de valores cujas chaves são inteiros positivos consecutivos começando por 1. No caso de atributos de equipamentos ou pontos cujo valor é uma lista, essas listas são especiais e só permitem leitura. Portanto, não é possível alterar os elementos de um atributo-lista. Contudo é possível substituir o valor do atributo-lista por completo por uma outra lista representada por uma tabela Lua.
