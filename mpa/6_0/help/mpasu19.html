6.2.2 Manipulando Elementos de Composição

Assim como em um editor gráfico, é possível selecionar e mover os elementos de composição de um diagrama. Para tanto, utiliza-se o botão Edita (cujo ícone é uma seta). Com este botão selecionado, também é possível realizar as seguintes operações:

Selecionar elementos de composição
: clique no elemento de composição desejado; marcadores quadrados são apresentados em suas extremidades, confirmando e habilitando o objeto para edição, como mostra a figura 6.4.


Figura 6.4: Marcadores identificam elemento de composição selecionado para edição.

Selecionar múltiplos elementos de composição
: clique e arraste o mouse de forma que o retângulo de seleção englobe todos os elementos a serem selecionados, como mostra a figura 6.5. Além disso, também é possível selecionar vários objetos clicando-os com a tecla control pressionada. Note que quando apenas um elemento está selecionado, aparecem pequenos quadrados nos vértices, indicando que o mesmo pode ser redimensionado; já quando dois ou mais elementos estão selecionados, o redimensionamento não é possível, então apenas a borda cinza fica visível, indicando que o grupo está selecionado e pode ser movido ou apagado.


Figura 6.5: Selecionando um grupo de elementos de composição.

Mover elementos de composição
: clique sobre o objeto a ser movido e, sem largar o botão, arraste o mouse até a posição desejada para o objeto; solte o botão para concluir a movimentação. A movimentação de grupo de objetos é possível através da seleção do grupo de objetos a serem movimentados e arrastando os mesmos em conjunto.
Redimensionar elementos de composição
: com o elemento de composição selecionado, clique sobre um marcador e, sem largar o botão, arraste-o até que o objeto fique do tamanho desejado, como mostra a figura 6.6; solte o botão para efetivar o redimensionamento.


Figura 6.6: Redimensionando um elemento de composição.

Apagar elementos de composição
: selecione o elemento de composição desejado e aperte a tecla Delete do teclado. Para apagar um grupo de objetos, basta que o grupo esteja selecionado antes de pressionar a tecla Delete.
Copiar e colar elementos de composição
: selecione o elemento de composição desejado (ou grupo de elementos e suas ligações) e aperte as teclas Ctrl e C, para copiar, ou Ctrl e X, para remover; isso fará com que uma cópia da seleção seja armazenada na área de transferência interna do MPA. Feito isso, basta apertar as teclas Ctrl e V para colar o conteúdo da área de transferência na área de trabalho. Note que cada colagem é deslocado de uma unidade de grid para direita e uma unidade de grid para baixo, para evitar que os elementos se sobreponham. Se a colagem estiver fora da área visível, esta será deslocada e até redimensionada, se necessário, para que a colagem seja exibida. A seleção sempre será colada desconectada do resto do diagrama, mesmo que fique posicionada sobre outros elementos. É possível copiar uma seleção de um diagrama para outro, contudo, apenas os elementos compatíveis com a linguagem visual do diagrama serão colados. A combinação Ctrl e D duplica automaticamente a seleção (é equivalente à sequência Ctrl e C, seguida de Ctrl e V, exceto pelo fato de não alterar o conteúdo da área de transferência).
Definir o conteúdo dos elementos de composição
: com o botão Edita selecionado, selecione o elemento e o painel contextual se adaptará para exibir os controles de configuração do elemento. A seção 6.3 apresenta os painéis de edição de cada um dos elementos de composição.

Manipulando Elementos de Ligação

Os elementos de ligação nos diagramas, tipicamente representados por linhas podem ser alterados desde que o objeto esteja selecionado:

  • Um vértice pode ser reposicionado livremente através de um clique-e-arrasto no vértice (note que a cor das arestas adjacentes muda para vermelha e durante o arrasto as novas arestas vão sendo desenhadas em azul)
  • Um novo vértice pode ser inserido com o uso da tecla shift em conjunto com um clique-e-arrasto no meio de uma aresta (note que somente a aresta clicada fica vermelha e as novas arestas – fruto do particionamento da original – são desenhadas em azul); ao soltar o botão do mouse, a posição do novo vértice é determinada
  • Uma aresta pode ser reposicionada livremente através de um clique-e-arrasto na aresta (note que a cor das duas arestas adjacentes muda para vermelha e durante o arrasto elas são desenhadas em azul);

O MPA simplifica automaticamente a situação em que duas arestas contíguas ficam alinhadas: elas são transformadas em uma única aresta e o vértice que as separa é removido. Esse comportamento vale tanto para o caso de reposicionamento de vértices quanto de arestas, quando essa simplificação pode ocorrer simultaneamente com três arestas e os dois vértices entre elas.
