C.1.3 Tipos de Equipamentos

O MPA organiza os diversos equipamentos manipulados em classes de equipamentos que apresentam características comuns (i.e. atributos). Cada classe de equipamento define um novo tipo de equipamento. Tanto atributos de equipamentos quanto variáveis do diagrama, valores de parâmetro ou valores de retorno de funções podem ter seu tipo como uma classe de equipamento. O campo type serve para indicar o tipo do atributo e deve conter sempre um identificador. Na próxima seção, será apresentado como é feita a definição de classes de equipamentos.
