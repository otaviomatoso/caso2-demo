11.2 Segurança

O servidor de execução permite a definição de uma senha de acesso como forma de restringir o acesso. Para definir uma senha de acesso utiliza-se o parâmetro de configuração password. Por exemplo, iniciar o servidor com a senha de acesso ”senhadeacesso”utiliza-se o seguinte comando:

mpa-run.exe --password senhadeacesso

Ou adiciona-se ao arquivo de parâmetros de configuração o seguinte:

PASSWORD = ”senhadeacesso”;

Caso nenhuma senha seja definida através desse parâmetro de configuração, então não é feita nenhuma restrição de acesso ao servidor de execução, ou seja, é possível conectar-se ao servidor sem definir nenhuma senha.
