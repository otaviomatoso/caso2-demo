6.6.3 Visualização de Execução

Quando uma linha de execução é selecionada na aba Depuração, os últimos elementos ativados por aquela linha de execução passam a ser marcados com contornos em branco. O último elemento ativado é sempre exibido com contorno em azul, como ilustrado na figura 6.43. Entretanto, se a linha de execução está em pausa quando a sessão de depuração for iniciada, então nenhum elemento é marcado como ativado até que a linha de execução prossiga. Portanto, se a há pausas muito longas na execução dos diagramas, possivelmente nenhum elemento do diagrama será marcado como ativado quando a sessão de depuração é iniciada, pois a linha de execução estará suspensa naquele momento.

Durante a visualização da execução, é possível realizar pequenas edições no diagrama que não alterem seu comportamento, tais como a disposição dos elementos ou formato das linhas. Entretanto, se a alteração alterar o comportamento definido pelo diagrama, então a sessão de depuração é encerrada, pois não é possível continuar o acompanhamento.

Além disso, o valor das variáveis disponíveis naquele diagrama são exibidas na aba Variáveis no painel contextual lateral, conforme é descrito abaixo. Sempre que a linha de execução sendo exibida é trocada, o fundo do editor de diagramas muda de cor rapidamente para sinalizar que uma nova linha de execução está sendo exibida.
