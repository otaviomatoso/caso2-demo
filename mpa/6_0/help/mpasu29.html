6.3.7 Conector

O elemento de composição Conector é representado graficamente por um círculo cinza. O Conector é usado principalmente para definir ligações entre elementos distantes no fluxo sem a necessidade de criar linhas longas que cortem ou contornem uma grande área do fluxo. Cada Conector possui uma descrição que o identifica. Todos os Conectores com a mesma descrição indicam o mesmo ponto de conexão no fluxo. Em outras palavras, todas as linhas que chegam a um Conector são implicitamente continuadas pelo Conector de mesma descrição que apresente uma linha de saída, como ilustrado na figura 6.30.


Figura 6.30: Uso do Conector como ligação descontinuada.

Nesse exemplo, tanto a saída da caixa Origem 1, quanto a saída da caixa Origem 2 têm como destino a caixa Destino, pois todos os Conectores utilizados possuem a mesma descrição e portanto definem o mesmo ponto. Quando um conector é selecionado, todos os demais conectores com a mesma descrição são exibidos em um tom de cinza mais claro para facilitar sua localização no fluxo. Em particular, o conector de saída é exibido em branco. Esse destaque é ilustrado na figura 6.31


Figura 6.31: Conector selecionado em destaque.

Não é permitido ter dois Conectores com mesma descrição que tenham uma ligação de saída, pois isso gera ambiguidade na continuação do fluxo após o Conector, conforme ilustrado na figura 6.32.


Figura 6.32: Conector com saída ambígua.

Também não é permitido definir Conectores com uma dada descrição sem que haja um com uma saída, pois nesse caso a continuação do fluxo é indeterminada.

Também é possível utilizar um Conector como um agrupador de linhas que possuem o mesmo destino, conforme ilustrado na figura 6.33. Nesse caso, inclusive pode haver outros Conectores que são usados para desviar a execução para o Conector agrupador.


Figura 6.33: Conector agrupando linhas convergentes.

Note que nesse caso o Conector também pode funcionar como um rótulo de um ponto específico do fluxo, como ilustrado na figura 6.34.


Figura 6.34: Conector como rótulo de um ponto do fluxo.
