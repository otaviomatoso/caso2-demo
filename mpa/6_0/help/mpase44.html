C.7 Carga Dinâmica de Módulos Adicionais

A versão do servidor MPA para plataformas Microsoft Windows oferece suporte para carga dinâmica de pacotes de Lua 5.12 através da função require. Inclusive é possível carregar pacotes escritos em C/C++. Entretanto, nesse último caso, é necessário que o módulo seja compilado como uma DLL compatível com o compilador do Microsoft Visual C++ 2003 (single-threaded DLL). Para maiores informações sobre pacotes de Lua 5.1, consulte o capítulo 15 do livro Programming in Lua3 .

A instalação padrão do servidor MPA para plataformas Microsoft Windows também inclui o pacote Alien4 , que permite acessar bibliotecas C/C++ a partir do código Lua definido no arquivo de módulo.
