B.3 Preferências de Vizualização

As figuras que representam configurações de equipamentos são, em geral, maiores que as que representam configurações de pontos, pois enquanto estas só podem ter atributos de tipos básicos, aquelas podem incluir atributos que são classes de pontos ou mesmo outras classes de equipamentos. Dessa forma, é possível que estas configurações fiquem realmente grandes, complicando sua visualização. Para simplificar a exibição nesses casos, o usuário pode configurar quais elementos serão visualizados. Isto pode ser feito através do menu Editar → Preferências.... Este diálogo apresenta uma área para configuração das referências inexistentes na planta e uma área para configuração da visualização. As opções são as seguintes:

Valores Simples
exibe os atributos dos tipos básicos, como números, valores lógicos e textuais.
Pontos de Controle
exibe os atributos que ainda não foram configurados.
Configurações Inexistentes
exibe configurações que são referenciadas, mas ainda não foram criadas ou não existem mais.
Atributos em Branco
mostra os pontos que são referenciados pela configuração.
Profundidade
define, para configurações que tenham muitas referências a outras configurações, o número máximo de níveis de referências que se deseja visualizar. Por exemplo, considere um equipamento que tem como atributo outro equipamento e esse tem como atributo um ponto. Se o nível de recursão estiver configurado para um, o ponto referido pelo segundo equipamento não será exibido no grafo.
