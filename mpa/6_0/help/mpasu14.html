5.2.5 Trabalhando com arquivos CSV

As abas de edição de instâncais de um classe de configuração de Planta permite que o usuário edite as planilhas das classes em outros editores de planilhas e exporte/importe estes dados usando o formato CSV (Comma Separatted Values - Valores Separados por Vírgulas).

A ferramenta de exportação/importação de arquivos CSV pode ser acessada a partir da Barra de Ferramentas localizada à direita da planilha de classe.

Para exportar a lista de configurações para um arquivo CSV basta selecionar a opção no menu de contexto e escolher ou definir o arquivo em que deve ser salvo.

A importação do arquivo CSV deve ser feita com atenção, pois interfere diretamente no projeto de planta. É uma boa prática salvar o projeto atual antes de realizar a importação, pois a mesma irá substituir a planilha atual pela planilha carregado do arquivo.
