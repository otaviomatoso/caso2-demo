7.1.1 Gerenciando Classes Estendidas

Para poder estender classes equipamentos ou pontos com novos atributos é necessário antes adicionar sua classe ao arquivo de extensões. Isso é feito através do editor do arquivo de extensões que exibe a lista de classes presentes no arquivo. Os botões laterais permitem adicionar e remover as classes do arquivo. Ao clicar no botão de adicionar classe é exibido um diálogo de seleção das classes existentes na aplicação, similar ao diálogo de adição de classes ao arquivo de plantas. Cada classe presente no arquivo de planta é exibida como um documento (nó filho) do arquivo na árvore de projetos. Para editar as configurações daquela classe no arquivo deve-se editar o documento correspondente.
