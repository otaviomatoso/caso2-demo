C.1.2 Tipos de Pontos de Controle

Além dos tipos de valores básicos, o MPA também define um tipo de ponto de controle para cada um dos tipos de valores descritos anteriormente. Esses tipos de pontos são automaticamente mapeados para os tipos dos pontos do sistema supervisório utilizado pelo MPA de forma transparente ao desenvolvedor de diagramas ou pré-configurador. Ou seja, se o sistema supervisório manipula tipos como inteiro com e sem sinal, números em ponto flutuante de precisão simples e dupla, todos esses tipos são automaticamente mapeados para os tipos básicos do MPA. Através desses tipos, o MPA permite especificar quais tipos de pontos de controle que um equipamento deve apresentar. Tomemos por exemplo, a classe de equipamentos Válvula Choke. Se for definido que essa classe apresente um atributo do tipo BOOLEAN_POINT que especifica o ponto de controle utilizado para fazer um pulso de abertura na válvula choke, então todas as instâncias da classe Válvula Choke devem definir um ponto de controle lógico que permita realizar esse comando na válvula através da manipulação deste. A seguir, é dada uma lista dos tipos de pontos de controle do MPA:

BOOLEAN_POINT
Ponto de controle que assume valores lógicos, ou seja, assume apenas os valores verdadeiro (true) e falso (false).
INTEGER_POINT
Ponto de controle que assume valores numéricos inteiros, ou seja, números sem parte fracionária.
REAL_POINT
Ponto de controle que assume valores numéricos reais, ou seja, números com parte fracionária.
STRING_POINT
Ponto de controle que assume valores textuais, ou seja, seqüências de caracteres.
