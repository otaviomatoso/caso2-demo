C.4 Definição de Funções Auxiliares

Funções auxiliares são definidas através do construtor func. Cada definição especifica o nome da função e um identificador. A definição de uma função auxiliar também inclui uma lista de parâmetros, contendo o nome e o tipo de valor de cada parâmetro. Além disso, a definição de uma função de equipamento também contém uma descrição e a definição de um valor de retorno, com nome e tipo. Os tipos dos parâmetros e valor de retorno das funções só podem ser tipos básicos ou equipamentos (definido através do identificador da classe de equipamento). A figura C.4 ilustra a definição de uma função auxiliar (de nome esperar) e de uma função não disponível para diagramas (de nome limite).


Figura C.4: Exemplo de definição de uma função auxiliar.
 
func { 
  id = ”esperar”, 
  name = ”Esperar”, 
  parameters = {{ name = ”Tempo de Espera”, type = ”INTEGER” }}, 
  description = ”Faz uma pausa na execucao do diagrama”, 
  code = ”sleep”, -- a implementação é a própria função ’sleep’ 
                  -- oferecida pelo ambiente de execução do MPA 
} 
 
code = [[ 
  function limite(valor, minimo, maximo) 
    return math.min(maximo, math.max(minimo, valor) ) 
  end 
]]
