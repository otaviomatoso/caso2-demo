6.2.1 Adicionando Elementos de Composição

Os elementos de composição de diagramas são adicionados utilizando-se os botões à esquerda da área de trabalho. Existe um botão para cada tipo diferente de elemento de composição. Para adicionar um novo elemento de composição, basta selecionar o botão correspondente ao elemento desejado e então clicar sobre a área de edição de diagramas. Também é possível criar o novo elemento com um tamanho determinado, bastando clicar e arrastar o mouse de forma a englobar a área onde o elemento deva ser criado1 . Neste caso, o elemento criado ocupará toda a área selecionada e o seu tamanho se torna o tamanho padrão dos próximos elementos criados daquele mesmo tipo.
