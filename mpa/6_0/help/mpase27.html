11.3 Biblioteca de Modelos Inicial

Ao executar o servidor de execução do MPA, é possível fornecer o arquivo de pré-configuração a ser carregado inicialmente pelo servidor. O local do arquivo de pré-configuração é especificado através do parâmetro de configuração preconfig. Como exemplo, suponha que se deseje iniciar o servidor de execução com o arquivo de pré-configuração C:\MPA\plantas\p31\main.mpam. Neste caso, pode-se utilizar o seguinte comando:

mpa-run.exe --preconfig C:\MPA\plantas\p31\main.mpam

Alternativamente, pode-se adicionar ao arquivo de parâmetros de configuração o seguinte:

PRECONFIG = ”C:\MPA\plantas\p31\main.mpam”;

Caso esse parâmetro não seja definido, o servidor é iniciado apenas com a definição das classes de ponto pré-definidas do MPA.
