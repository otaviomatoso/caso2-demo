Capítulo 7
Atributos Estendidos

Este recurso visa permite estender as classes definidas na pré-configuração de forma que apresentem novos atributos para serem usados em diagramas. Esses atributos são voltados especificamente para execução dos diagramas. Dessa forma, não é necessário alterar a pré-configuração para se ajustar às necessidades de cada nova aplicação desenvolvida no MPA. Essa extensão se resume apenas a atributos, não é possível definir novos métodos de classe através do projeto de diagramas.
