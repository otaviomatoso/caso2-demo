5.2.3 Alterando valores de atributos de configurações

A edição de valores de atributos de configurações é feita editando os campos das células correspondentes da Planilha de Configurações.

Assumindo que o usuário já selecionou a classe em que se encontra a configuração que deseja editar, os atributos da classe estão carregados na Planilha de Configurações. Definida a linha em que se encontra a configuração a editar basta clicar duas vezes na célula correspondente para editar o valor.

Quando o atributo é uma lista de valores, os valores da lista são exibidos no painel contextual como ilustrado na figura 5.3. Através dessa lista no painel contextual, é possível adicionar novos valores a lista, remover valores da lista, mudar a ordem dos valores da lista, assim como alterar os valores já contidos na lista.


Figura 5.3: Diálogo de edição de atributos-lista.

Se o atributo editado for um equipamento ou um ponto ou ainda uma lista de equipamentos ou pontos e o valor indicado não seja o nome de uma configuração, o programa irá identificar isso e poderá adotar diferentes ações de acordo com as preferências do usuário quanto a verificação de nomes, que podem ser alteradas através do menu Editar → Preferências..., como apresentado na seção 3.2.2. De acordo com as preferências atuais, o programa pode adicionar a uma configuração com o nome indicado na classe adequada.

Se o valor digitado não estiver de acordo com o tipo do atributo correspondente, o texto da célula ficará vermelho.

Edição Múltipla É possível editar mais de um atributo de uma vez. Contudo isso não é possível para atributos que são uma lista. Nesse caso deve-se selecionar as células utilizando o cursor do mouse e as teclas Control e Shift, podendo definir um grupo de células não-contínuo. Com todas as células que terão o novo valor selecionadas basta selecionar a opção Editar do menu de contexto e digitar o novo valor no diálogo que será exibido.
