6.2 Editor de Diagramas

A elaboração dos diagramas é feita no editor de diagramas da IDE, que é ilustrado na figura 6.3. O editor apresenta uma barra vertical de botões e uma área para edição de diagramas, através dos quais é possível criar e manipular elementos de composição de diagramas como em um editor gráfico. A área de edição de diagrama apresenta uma grade para facilitar o alinhamento dos elementos de composição de diagramas.


Figura 6.3: Edição de diagrama de fluxograma.

O editor de diagramas funciona de forma similar a um editor gráfico. Neste manual, utilizaremos o termo elemento de composição para designar os elementos gráficos utilizados para construir os diagramas do MPA. Os elementos de composição são representados por figuras geométricas, tais como retângulos, losangos, círculos ou setas. Cada linguagem visual do MPA apresenta um conjunto de elementos de composição diferentes e cada elemento possui uma semântica diferente. A seção 6.3 descreve detalhadamente a semântica de cada um dos elementos de composição de diagramas do MPA.
