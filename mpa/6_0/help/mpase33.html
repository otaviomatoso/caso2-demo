A.1 Funções disponibilizadas

Para dar suporte ao desenvolvimento de scripts, foi desenvolvida uma API com funções que permitem o acesso aos dados do arquivo de projeto corrente de planta ou diagramas.

MPA.PrintOut( ... )
Imprime no campo de saída os parâmetros passados para a função.
MPA.Message( msg )
Abre uma janela com o título “

”Informação com o conteúdo msg;

MPA.Option( title, msg, first, second, third )
Abre uma janela com o título “

”title com o conteúdo msg e botões com rótulos: first, second, third. Retorna o número da opção selecionada ;

MPA.LoadFileDialog(title,filter,info)
Exibe um diálogo para seleção de arquivo para abrir com título title, exibindo apenas arquivos com extensão filter, e apresentando descrição para filtro info. Retorna o caminho para o arquivo selecionado ou nil se a operação for cancelada;
MPA.OpenDirDialog(title)
Exibe diálogo para seleção de diálogo com título title. Retorna o caminho para o diálogo selecionado ou nil se a operação for cancelada;
MPA.SaveFileDialog(title,filter, info)
Exibe um diálogo para seleção de arquivo para salvar com título title, exibindo apenas arquivos com extensão filter, e apresentando descrição para filtro info. Retorna o caminho para o arquivo selecionado com extensão aplicada ou nil se a operação for cancelada;
MPA.DisableNameCheck()
Desabilita a verificação de referências ;
MPA.EnableNameCheck()
Habilita a verificação de referências ;
MPA.RecoverNameCheck()
Recupera modo verificação de referências anterior a chamada de MPA.DisableNameCheck() ou MPA.EnableNameCheck();
MPA.AddConfig( classid, cfgid )
Adiciona uma configuração vazia na classe classid com o nome cfgid, a configuração pode ser tanto de ponto quanto de equipamento, desde que o identificador da classe (classid) represente a classe desejada;
MPA.RemoveConfig(tag, classid)
Remove a configuração identificada por tag da classe identificada por classid ;
MPA.SetTag(c, tag, classid)
Atribui o identificador tag a configuração c da classe identificada por classid;
MPA.ConfigExists(name, classid)
Função para informar se uma configuração com identificador name existe na classe identificada por classid ;
MPA.GetAttribute(attridx, classid)
Função que retorna definição do atributo identificador por attridx na class identificada por classid ;
MPA.SetAttribute(classid, cfgid, attrid, val)
Define o valor val ao atributo identificado por attrid da configuraração identificada por cfgid da classe identificada por classid ;
MPA.GetAttributeName(classid, attridx)
Retorna o nome do atributo de índice attridx da classe identificada por classid ;
MPA.GetAttributeType(classid, attridx)
Retorna o tipo do atributo de índice attridx da classe identificada por classid ;
MPA.GetAttributeAccess(classid, attridx)
Retorna o tipo de acesso do atributo de índice attridx da classe identificada por classid ;
MPA.ExportCSV(filename, classid, append)
Exporta a classe identificada por classid para o arquivo filename. Se o parâmetro append for true adiciona o conteúdo ao arquivo, caso contrário sobreescreve o conteúdo atual do arquivo;
MPA.ImportCSV(filename, classid, append)
Importa o conteúdo do arquivo filename para a classe identificada por classid. Se o parâmetro append for true adiciona o conteúdo do arquivo a classe, caso contrário sobreescreve o conteúdo atual da classe;
MPA.GetDiagramNames()
Retorna uma tabela com os nomes dos diagramas configurados no arquivo de projeto de fluxos.
MPA.AllNodes(diagram)
Função para iterar sobre todos os elementos do diagrama diagram;
MPA.AllLinks(diagram)
Função para iterar sobre todos os elementos conectores do diagrama diagram;
MPA.AllAnnotations(diagram)
Função para iterar sobre todos os elementos de anotação do diagrama diagram;
MPA.BuildSequence(flow)
Função usada para construir a seqüência entre os nós de um fluxo. Permite a navegação seqüencial dos nós a partir das funções MPA.GetStartPoint e MPA.GetNthPath.
MPA.GetStartPoint(seq)
Retorna o ponto de início de uma sequencia de elementos de um fluxo.
MPA.GetNthPath(node, n)
Retorna o enésimo caminho de saída de determinado nó do fluxo. Além do caminho de saída de um Comando, pode ser usado para obter os caminhos de saída alternativos dos elementos Iterador, Escolha ou Executor.
MPA.NodeType(node)
Retorna uma string com o tipo do nó dado.
MPA.ActionType(node)
Retorna uma string que representa o tipo da ação definida para o nó.
MPA.GetDescription(node)
Retorna a descrição definida para o nó.
MPA.GetInfo(node, kind)
Essa função é usada para obter informações detalhadas do nó. A informação desejada deve ser passada no parâmetro kind de acordo com a seguinte lista de códigos:
  • ”iterated.class” - classe para a qual a ação do nó é definida;
  • ”message.pre” - mensagem de acompanhamento anterior;
  • ”message.post” - mensagem de acompanhamento posterior;
  • ”diagram.name” - nome do diagrama que é chamado;
  • ”diagram.action” - ação desenvolvida sobre o diagrama chamado;
  • ”expression” - expressão definida para escolha;
  • ”function.id” - identificador da função auxiliar chamada;
  • ”parameters” - parâmetros passados para função auxiliar ou método;
  • ”results” - variável em que os retornos da chamada de função, método ou atribuição serão armazenados.
  • ”method.class” - classe do método chamado;
  • ”method.id” - identificador do método chamado;
  • ”equipment.variable” - variável de equipamento sobre a qual o método será chamado;
  • ”equipment.ids” - identificadores fixos dos equipamentos sobre os quais o método será chamado.
