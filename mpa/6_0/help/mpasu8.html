3.4.2 Documentos em Arquivos

Muitos arquivos apresentam definições internas, denominadas documentos, que são apresentadas como nós filho do arquivo na árvore do projeto. Exemplos de documentos são diagramas individuais em uma biblioteca de diagramas, ou instâncias de uma classe em uma planta de equipamentos. Os documentos individuais também são abertos para edição através de um clique-duplo sobre o nó que o representa.

A árvore de projetos também permite copiar definições de um arquivo para outro do mesmo tipo. Por exemplo, para copiar um diagrama para outro arquivo de biblioteca de diagramas basta arrastar o nó da árvore que representa o diagrama para o nó que representa no arquivo de destino. De forma análoga, para mover esse documento de um arquivo para o outro, basta arrastar o nó com a tecla Shift pressionada.

Outra funcionalidade disponível na árvore para documentos é a habilitação de documentos do arquivo. Por exemplo, seu um diagrama ainda está incompleto e não deve ser exportado para arquivo ou enviado para o executor, o usuário pode clicar com o botão direito sobre o nó que corresponde ao documento do diagrama e selecionar o item Desabilitar. Quando esse diagrama estiver completo, basta repetir o clique no documento e selecionar o item Habilitar.
