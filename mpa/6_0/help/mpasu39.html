6.6.1 Aba de Depuração

Após a seleção dos diagramas-aplicação, a aba de depuração ilustrada na figura 6.42 é exibida no painel inferior. Essa aba exibe uma barra com cinco botões e uma tabela com as linhas de execução sendo observadas do diagrama, conforme descrito a seguir:


Figura 6.42: Aba de depuração no painel inferior.

Botões de Adição e Remoção de Linhas de Execução Os dois primeiros botões permitem adicionar e remover da tabela de linhas de execução sendo observadas, respectivamente. Apenas as linhas de execução sendo observadas podem ser acompanhadas visualmente no diagrama durante a depuração.

Botões de Controle da Execução Esses botões só ficam habilitados quando alguma linha de execução atinge um ponto de parada (veja a seção 6.6.4). O primeiro botão (símbolo de play) é o comando Continuar Execução, que permite continuar a execução até o próximo ponto de parada.

Os dois botões mais abaixo executam os comandos Passo da Linha Atual e Passo de Execução e servem para fazer o servidor executar um passo. Esse passo pode considerar apenas a própria linha de execução ou também incluir as outras linhas de execução concorrentes. Sendo assim, a ideia é que o Passo da Linha Atual sirva para seguir a linha de execução corrente, indo para a caixa seguinte, que pode estar em outro diagrama se o comando corrente for uma chamada de diagrama-função. A ideia do Passo de Execução é considerar outras linhas de execução, indo para a caixa de algum outro diagrama, que está interrompido e pertence a outra linha de execução. Porém, isso só acontece quando o comando corrente provoca algum tipo de espera6 , o que dá a chance do executor de diagramas trocar de linha de execução. Portanto, se o comando corrente não provoca nenhuma espera, os dois botões produzem sempre o mesmo efeito.

Há ainda uma situação que merece ser enfatizada. A primeira é quando o comando corrente provoca uma espera e o servidor encontra um ponto de parada em alguma outra linha de execução. Nesse caso, independentemente de qual botão foi pressionado, a execução será sempre interrompida onde o outro ponto de parada foi encontrado.

Tabela de Linhas de Execução Essa tabela permite visualizar as linhas de execução dos diagramas aplicação disponíveis para depuração. Quando uma linha de execução é adicionada a essa tabela, é possível acompanhar sua execução fazendo um clique-duplo na coluna Diagrama correspondente à linha de execução desejada. Também é possível renomear o rótulo da linha de execução para um texto mais descritivo fazendo um clique-duplo na coluna Rótulo.
