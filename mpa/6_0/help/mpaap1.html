Criação de Scripts de Comando

O MPA oferece um conjunto de scripts para a automação de tarefas referentes à IDE. Estes scripts manipulam um conjunto de opções bem próximo à utilizado internamente pela prória IDE, tendo acesso às estruturas de dados e alguns procedimentos disponíveis na interface de maneira programática.

O diálogo de seleção de script (figura A.1) é exibido selecionando o menu Arquivo → Executar script....

O diálogo de execução de scripts apresenta uma lista para escolha do script (no alto), seguido do nome do autor e da descrição do mesmo. Na parte de baixo do diálogo, com o título saída de execução, há um campo de texto somente de leitura que é onde são exibidas as mensagens de execução do script. Logo abaixo desta área, vêm os botões Executar, Gravar e Fechar. O botão Gravar exibe um diálogo para deifnição de nome e caminho do arquivo em que será gravado o resultado da última execução de script.

Dentre os scripts pré-definidos, os mais relevantes são os seguintes:

  • Relatório do Projeto de Fluxos Corrente - cria um relatório com a configuração dos elementos de fluxograma seqüencialmente. O script exibe todos os elementos dos fluxogramas do projeto corrente.
  • Geração de descrição de planta em arquivo html - cria uma página na linguagem HTML da descrição da planta com a lista de classes de equipamento, lista de configurações das classes e lista de atributos dos equipamentos. A página apresenta um código de estilos css (Cascading Style Sheets) verde e amarelo e devido à sua simplicidade pode ser visualizada em quase qualquer Navegador de Internet.
