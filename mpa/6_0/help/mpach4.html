Capítulo 4
Bibliotecas de Modelos

Um arquivo de biblioteca de modelos é um arquivo escrito na Linguagem Lua que define modelos no ambiente de execução de aplicações MPA. Esse modelos podem ser funções auxiliares ou classes de equipamento ou pontos.

Uma função auxiliar é uma implementação de uma função Lua que é disponibilizada para ser chamada diretamente por um dos diagrama que compõem a aplicação MPA. Essas funções são descritas com um número fixo de parâmetros e valores de retorno. Os tipos dos valores dos parâmetros e valores de retorno das funções auxiliar é determinado pelo próprio arquivo ou pelas bibliotecas de modelos incluidas pelo mesmo.

Uma classe de equipamento basicamente define os tipos dos pontos de controle que permitem manipular um determinado tipo de equipamento na aplicação. Por exemplo, a classe de equipamento Válvula On/Off pode definir que esses equipamentos são manipulados por três pontos de controle que assumem valores lógicos (BOOLEAN_POINT), onde um indica que a válvula está completamente aberta, outro indica que está completamente fechada e terceiro indica se a válvula está em falha. A classe de equipamento também pode definir outros tipos de valores para exprimir outras características dos equipamentos, como um valor inteiro para indicar o tempo em segundos para a movimentação completa de cada válvula. Finalmente, uma classe de equipamento também pode definir operações para manipular o equipamento. Essas operações sobre equipamentos são descritas de forma similar às funções auxiliares. Contudo, essas operações sempre são executadas sobre um equipamento da classe.

Uma classe de pontos é similar a uma classe de equipamento, contudo ela necessariamente estende uma das classes de ponto pré-definidas do MPA e tem acesso direto a uma ponte de comunicação para acessar o tag correspondente ao ponto de controle.

Na versão atual do MPA, os arquivos de módulos são editados fora da IDE, tipicamente num editor de texto ou de código Lua. O apêndice C apresenta o formato dos arquivos de módulo.
