C.5.2 Funções do Executor

O ambiente de execução de diagramas MPA, fornece um conjunto de operações que podem ser utilizadas para realizar operações como interromper a execução por um período de tempo, ou consultar se um diagrama está em execução ou não. A seguir, é dada uma lista de todas as operações fornecidas pelo ambiente de execução do MPA para auxiliar a implementação de funções auxiliares ou funções sobre equipamentos.

instancesof(classid)
Permite obter um iterador a ser utilizado num for que permite iterar sobre todas as instâncias da classe com o identificador classid. As variáveis de iteração recebem respectivamente o identificador da instância (i.e. equipamento ou ponto) e a instância propriamente dita.
getinstance(classid, id)
Permite obter a instância cujo identificador é id da classe com o identificador classid.
getdiagrams()
Permite obter uma lista contendo os nomes dos diagramas presentes no executor.
diagramstatus(id)
Devolve uma string informando o estado do diagrama de nome id. Os valores possíveis são ”running” e ”stopped”. Caso o diagrama informado não exista, é devolvido o valor nil e uma mensagem de erro.
startdiagram(id)
Permite agendar o diagrama de nome id para execução. Essa função devolve o valor true caso o diagrama esteja pronto para ser executado. Caso contrário é devolvido nil e uma mensagem de erro.
stopdiagram(id)
Permite encerrar a execução do diagrama de nome id. Essa função devolve o valor true caso o diagrama seja encerrado. Caso contrário é devolvido nil e uma mensagem de erro.
getpoint(tag)
Permite acessar diretamente o valor do ponto de controle tag no supevisório através da ponte de comunicação. Esta função devolve o valor do ponto de controle, ou o valor nil seguido de uma mensagem de erro.
setpoint(tag, value)
Permite alterar o valor do ponto de controle tag para o valor value no supevisório através da ponte de comunicação. Esta função devolve true em caso de sucesso, ou o valor nil seguido de uma mensagem de erro em caso de falha.
getbridge(tag)
Permite obter a ponte associada a um ponto dado seu tag ou, se nenhum tag for fornecido, a ponte default é devolvida, pois é definida para todos os pontos que não casam com nenhum padrão definido pelas demais pontes.
sleep(time)
Permite suspender a execução de um diagrama pelo período de tempo definido pelo parâmetro time em segundos.
time()
Permite obter um contador de segundos do sistema. O valor devolvido é um valor real cuja precisão varia entre sistemas mas é sempre maior que milisegundos.
