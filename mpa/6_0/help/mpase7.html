3.3 Barra de Botões

A barra de botões principal contém botões para alguns itens do menu de comando que são mais comumente usados. Os botões disponíveis são:

Ícone Item do menu de comandos
Arquivo → Novo
Arquivo → Importar
Arquivo → Salvar Todos...
Editar → Desfazer
Editar → Refazer
/Executor → Conectar/Desconectar
Executor → Encerrar
Executor → Desbloquear
Executor → Enviar Aplicação
Executor → Apagar Aplicação
/Executor → Depurar Aplicação/Encerrar Depuração da Aplicação
Executor → Remover Bloqueios
