3.4 Árvore de Projeto

A Árvore de Projeto exibe os arquivos que compõem o projeto de aplicação atual. A pasta Bibliotecas de Modelos exibe os arquivos de bibliteca de modelos carregados e contém os documentos com definições de modelos de funções auxiliares e classes de ponto e equipamentos. A pasta Configurações de Plantas exibe os arquivos de configuração de planta do projeto de aplicação e os arquivos dos quais o projeto depende. Esses arquivos contém os documentos com instânicas de equipamentos e pontos disponíveis para o projeto de aplicação. A edição e descrição detalhada dessas instâncias são apresentados no capítulo 5. A pasta Extensões de Classes exibe os arquivos de extensões de classe do projeto de aplicação e os arquivos dos quais o projeto depende. Esses arquivos contém documentos com as definições de atributos estendidos para classes carregadas para o projeto. A edição e descrição de extensões de classes e atributos estendidos são apresentados no capítulo 7. A pasta Biblioteca de Diagramas exibe os arquivos de biblioteca de diagramas do projeto de aplicação e os arquivos dos quais o projeto depende. Esses arquivos contém documentos com Fluxogramas e SFCs disponíveis para o projeto de aplicação. A edição e descrição detalhada desses documentos são apresentados no capítulo 6. Arquivos que não são reconhecidos pela IDE são apresentados na pasta Outros.
