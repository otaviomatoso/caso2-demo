7.2 Editor de Atributos Estendidos

Ao editar uma classe de equipamentos ou pontos de um arquivo de extensões é exibido um editor que apresenta uma tabela com os atributos estendidos daquela classe, conforme ilustrado na figura 7.1.


Figura 7.1: Editor de atributos estendidos.

O editor apresenta uma tabela com os atributos estendidos daquela classe. A primeira coluna indica o nome do atributo, que é utilizado na geração dos métodos automáticos de acesso aos atributos estendidos. A segunda coluna indica o tipo do atributo. A quinta coluna só é ativa para atributos cujo tipo seja uma classe de ponto (não-lista) e permite informar quais operações sobre esse ponto estão ativas (leitura ou escrita). O valor de qualquer atributo estendido pode ser obtido e definido no diagrama, ou seja, é como se todos os atributos estendidos já apresentassem implicitamente o acesso gs. Os atributos estendidos de uma classe são herdados por todas as suas subclasses.
