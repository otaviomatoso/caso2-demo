5.1.1 Gerenciando Classes

Para poder configurar equipamentos ou pontos é necessário antes adicionar sua classe ao arquivo de planta. Isso é feito através do editor do arquivo de plantas que exibe a lista de classes presentes no arquivo. Os botões laterais permitem adicionar e remover as classes do arquivo. Ao clicar no botão de adicionar classe é exibido um diálogo de seleção das classes existentes na aplicação, conforme ilustrado na figura 5.1.


Figura 5.1: Diálogo de adição de classes em arquivo.

Cada classe presente no arquivo de planta é exibida como um documento (nó filho) do arquivo na árvore de projetos. Para editar as configurações daquela classe no arquivo deve-se editar o documento correspondente.
