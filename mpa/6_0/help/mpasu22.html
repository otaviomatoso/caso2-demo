6.2.5 Expressões

Vários elementos de composição dos diagramas utilizam expressões para definição de valores. As expressões no MPA podem conter números na forma decimal, os valores lógicos true e false e também textuais que devem ser envolvidos por aspas simples ou duplas, como em "texto" e ’texto’. Também é possível utilizar operadores aritméticos (+, -, *, /, ^), lógicos (and, or, not) e relacionais (>, <, >=, <=, ==, ~=) nessas expressões. Além disso, as funções matemáticas descritas em C.5.1 também podem ser utilizadas nas expressões.

Uma expressão também pode criar listas de valores, para tanto deve-se utilizar chaves em torno da sequência de valores, que devem ser separados por vírgula ou ponto-e-vírgula, como em {1,2,3}. Inclusive, é possível utilizar como elementos da lista expressões complexas, como em { 1.234^2, sin(3.14), atan(1.57)+7.8 }. Para criar uma lista vazia utilize {}.

As expressões também podem conter variáveis definidas no diagrama ou criadas pelo elementos de composição do diagrama, como será apresentado mais adiante. Quando uma variável contém uma lista, é possível referenciar um elemento específico da lista através da indexação de usa posição na lista. Por exemplo, se primos é uma lista com os 10 menores números primos, é possível obter o terceiro número através da expressão primos[3]. Esse tipo de indexação pode ser utilizada em expressões mais complexas, como por exemplo como o parâmetro de uma função matemática ou em conjunto com operadores.

Para obter o número de elementos numa lista, utilize a função length que recebe uma lista como parâmetro e devolve o número de elementos na lista.
