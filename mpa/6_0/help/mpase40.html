C.3 Definição de Classes de Pontos

Também é possível definir novas classes de ponto além das pré-definidas para cada tipo básico. A definição de classes de ponto é feita de forma idêntica à definição de classes de equipamento, inclusive através do mesmo construtor class. Entretanto, as classes de ponto devem herdar de outra class de ponto, como por exemplo as classes de ponto padrão definidas no MPA. Dessa forma, a classe implicitamente herda uma implementação que permite com que cada ponto dessa classe possa acessar a sua ponte de comunicação através do campo self.bridge. A figura C.3 ilustra a definição de uma classes de ponto que faz tradução para unidade de engenharia de acordo com valores definidos na configuração da planta.


Figura C.3: Exemplo de definição de classe de ponto.
 
class{ 
  id = ”ENG_UNIT_POINT”, 
  name = ”Ponto em unidade de engenharia”, 
  description = ”Ponto cujo valores são convertidos para unidade de engenharia”, 
 
  bases = { ”REAL_POINT” }, 
 
  attributes = { 
    { id = ”maxUnidEq”, 
      name = ”Máximo da Unidade do Equipamento”, 
      type = ”INTEGER” }, 
    { id = ”minUnidEq”, 
      name = ”Mínimo da Unidade do Equipamento”, 
      type = ”INTEGER” }, 
    { id = ”maxUnidEng”, 
      name = ”Máximo da Unidade de Engenharia”, 
      type = ”REAL” }, 
    { id = ”minUnidEng”, 
      name = ”Mínimo da Unidade de Engenharia”, 
      type = ”REAL” }, 
 
    ... 
  }, 
 
  methods = { 
    { id = ”read”, 
      name = ”Ler”, 
      results = {{ type = ”REAL”, name = ”Valor Corrente” }}, 
      description = ”Lê o valor atual do ponto”, 
      code = [[function(self) 
        local value = assert(self.bridge:get(tostring(self))) 
        value = (value-self.minUnidEq)/ 
                (self.maxUnidEq-self.minUnidEq) 
        return self.minUnidEng + 
               value*(self.maxUnidEng-self.minUnidEng) 
      end]] }, 
    { id = ”write”, 
      name = ”Escrever”, 
      parameters = {{ type = ”REAL”, name = ”Novo Valor” }}, 
      description = ”Escreve o novo valor do ponto”, 
      code = [[function(self, value) 
        value = (value-self.minUnidEng)/ 
                (self.maxUnidEng-self.minUnidEng) 
        assert(self.bridge:set(tostring(self), 
          self.minUnidEq + value*(self.maxUnidEq-self.minUnidEq) ) 
      end]] }, 
  }, 
}
