Apêndice B
Visualização de configurações

Configurar uma planta inteira na IDE pode ser uma tarefa demorada, mas criar cada planilha num editor comum que não tenha as funcionalidades de verificação da aba Planta pode dar margem a pequenos erros e falhas que comprometam o funcionamento das aplicações MPA.

Como já foi mencionado no capítulo 5, a aba Planta apresenta, dentre outras, uma ferramenta que permite visualizar os valores dos atributos das configurações e em alguns casos as relações entre estas.

O funcionamento desta ferramenta depende do software Graphviz incluido no instalador do Sistema MPA. A visualização das configurações só será possivel se o programa localizar o executável do Graphviz.
