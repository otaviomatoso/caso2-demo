6.3.3 Escolha

O elemento de composição Escolha permite que se ramifique o fluxo dependendo de uma certa condição lógica, isto é, de um resultado cujo valor pode ser verdadeiro ou falso. A Escolha é representada por um losango azul como na figura 6.16.


Figura 6.16: As ramificações V e F determinam a sequência do fluxo após a Escolha.

A decisão do objeto Escolha é feita pelo resultado de uma ação similar a executada pelos elementos Comando. Portanto essa decisão pode ser com base na avaliação de uma expressão, ou pelo resultado de uma chamada de função auxiliar, ou operação sobre um ponto ou equipamento, ou até chamada de diagrama-função. Essas alterativas são apresentadas no diálogo de seleção de ação que é exibido clicar no botão de definição de ação no painel contextual ao selecionar um elemento Escolha. Em qualquer um dos casos, o resultado final da condição precisa ser do tipo lógico, portanto apenas funções, operações sobre pontos ou equipamentos e diagramas-função que retornem um valor lógico são listados no diálogo. Isso é feito para evitar que o usuário possa selecionar uma opção inválida para o executor de diagramas. A seguir são apresentados os detalhes dessas opções.

Ação sobre Diagramas, Funções e Equipamentos

Os outros três tipos de configuração para o elemento Escolha são muito semelhantes entre si e aos correspondentes do elemento de composição Comando (veja seção 6.3.2). A diferença é que, no caso do elemento Escolha, as chamadas têm sempre que retornar um valor do tipo lógico, valor este que é usado para a tomada de decisão pelo executor do fluxo. Sendo assim, os elementos de interface para definição dos valores de retorno não são exibidos.

Note o exemplo da figura 6.18 que ilustra a chamada de uma operação sobre um equipamento cuja referência está armazanada na variável v.


Figura 6.18: A operação sobre um equipamento.

As ramificações a partir da escolha são feitas através do elemento de composição Ligação, sendo que é necessário definir uma descrição específica para determinar qual ramo é executado caso o resultado seja verdadeiro ou falso. Para o resultado verdadeiro, a ligação deve ter a descrição V e para o resultado falso a descrição deve ser F.
